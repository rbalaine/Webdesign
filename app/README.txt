Description de l'application Poste Finder :
Cette appliation ous permet de localiser la boite au lettre la plus proche de chez vous mais aussi la plus proche de votre position actuelle.

Réponses aux points clés du cahier de charges :
- Le site utilise des classes bootstrap afin d'avoir une application responsive qui s'adapte à tous les types de terminaux utilisés.
- Le css utilisée est un fichier scss dans lequel sont utilisés des fonctionnalités de SASS, à savoir, des variables, du mixing et du nesting
- L'application met en oeuvre le framework angularJS afin de manipuler des données de source libre. Les fonctionnalités utilisées sont les filtres, les routeProvider, les directives et l'api pour google maps.
- L'application est lançable facilement par l'utilisation de la commande grunt serve.

Problèmes rencontrés :
- L'api Angular pour google maps est très puissante mais la documentation est très pauvre car elle ne comprend aucun exemple d'utilisation
- Rendre l'application responsive pour toutes les tailles d'écran n'est pas toujours évident

Perspectives de développement :
L'application pourrait dans l'avenir mettre en place plus de fonctionnalités venant de google maps.

A noter que la localisation ne fonctionne que après un double click sur le boutton "Localisez moi"
